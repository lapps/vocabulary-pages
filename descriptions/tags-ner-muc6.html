Named entity category set used by the Sixth Message Understanding Conference (MUC-6), see https://cs.nyu.edu/faculty/grishman/muc6.html. This set includes three categories: PERSON, ORGANIZATION adn LOCATION.

In LAPPS, the Lingpipe Named Entity Recognizer service employs the ne-en-news-muc6.AbstractCharLmRescoringChunker model (see http://alias-i.com/lingpipe/web/models.html) which uses the MUC-6 category set.
